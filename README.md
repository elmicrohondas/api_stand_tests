# API Stand Tests

## Descripción del proyecto
Este proyecto contiene pruebas automatizadas para validar la funcionalidad de creación de usuarios en la API de Urban Grocers.

## Tecnologías utilizadas
- Python 3.x
- pytest
- requests

## Estructura del proyecto
```
api_stand_tests/
├── configuration.py      # Configuración de URLs y rutas
├── data.py              # Datos de prueba
├── sender_stand_request.py  # Funciones para enviar solicitudes HTTP
├── create_user_test.py  # Pruebas para creación de usuarios
├── .gitignore          # Archivos a ignorar por Git
└── README.md           # Este archivo
